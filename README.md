# Cómo construir un producto de IA Generativa en menos de 48 horas

Charla para **Desafío IA Bagó Perú 2026** · UPC Campus Monterrico · viernes 12 de junio.

Construida con [Slidev](https://sli.dev/).

## Desarrollo

```bash
npm install
npm run dev
```

Abre [http://localhost:3030](http://localhost:3030).

## Build / Export

```bash
npm run build    # estático en dist/
npm run export   # PDF
```

## Deploy

Push a `main` → GitHub Actions publica en [https://moshe-exe.github.io/desafio-ia-bago-peru-2026/](https://moshe-exe.github.io/desafio-ia-bago-peru-2026/).
